我们是谁

专注于「命令行里的生产力」

JoJo Code 是一个面向开发者的 Code CLI。我们的目标很简单：让你在终端里完成更多工作， 更少切换窗口。你可以把报错、代码片段、需求描述直接贴进 CLI，得到可执行的建议与下一步操作。

npm 全局安装
/connect 绑定 Key
/model 选择模型
我们的故事

从一个脚本开始，到一个 CLI

JoJo Code 起步于一个很小的脚本：把团队里常用的「代码审阅清单」和「排错流程」变成终端里的固定套路。 随着使用场景变多，它逐渐演化成一个可交互的 CLI：支持连接 Key、选择模型，并把常见工作流串起来。

同时也要感谢开源项目 Opencode，JoJo Code 的早期实现参考并基于其工程实践，我们在此之上做了适配、体验优化与功能扩展。

我们一直坚持一个原则：让工具更像“基础设施”，而不是又一个复杂平台。
