使用文档

最小可用：安装、连接、选模型。你只需要学会 /connect 和 /model。

查询额度 返回首页

/model 选择模型
Quick Start npm / connect / model
npm install -g jojocode

1) 安装

JoJo Code CLI 通过 npm 发布。建议使用 Node.js 18+。

npm install -g jojocode

2) 连接你的 Key

在 CLI 中输入 /connect 开始连接流程。完成后即可在会话里使用你的 Key。

连接步骤
  1. 1
    选择 Provider
    Anthropic OpenAI Google
  2. 2
    选择录入方式
    选择 Manually enter API Key
  3. 3
    粘贴 API Key 并完成
    连接成功后即可开始对话与选模型。

提示：Key 与使用额度相关，请妥善保管，避免泄露或转发给第三方。

jojocode /connect

3) 选择模型

输入 /model 选择当前会话模型；随时切换以匹配任务（例如更强推理/更快响应/更低成本）。

/model

常用提示

  • 把报错信息与相关代码片段一起贴上，能显著提高定位效率。
  • 给出目标与约束（语言/框架/风格/不能改的文件），输出会更稳定。
  • 需要可执行的修改时，说明你希望“输出补丁/给出修改步骤”。

额度与查询

你购买的是使用额度。需要查看剩余额度与使用记录，可前往查询页面。

打开查询统计 联系客服

常见问题

连接失败或提示无效 Key？

检查是否复制了完整 Key（无多余空格/引号），并确认该 Key 仍然有效；如仍无法解决，请邮件联系客服并附上错误截图。

如何切换模型？

在会话中输入 /model，按提示选择即可。切换后对后续对话生效。
